# DIU AI Hackathon 2026 Team Showcase

A premium React + Vite landing page for introducing mentors and developers of DIU AI Hackathon 2026.

## Run

```bash
npm install
npm run dev
```

Open the URL shown in terminal, usually:

```txt
http://localhost:5173
```

## Edit Data

- Mentors: `src/components/MentorSection.jsx`
- Developers: `src/components/DeveloperTeam.jsx`
- Logo/background images: `public/images/`
